# 🛡️ Kesedaran Bahaya Hacker & Keselamatan Server

Projek ini dibina untuk memberi pendedahan kepada masyarakat dan pentadbir sistem tentang risiko serangan siber dan bagaimana kod pengaturcaraan digunakan dalam eksploitasi.

## 🚀 Ciri-ciri Website
* **Penerangan Impak:** Kesan serangan kepada infrastruktur server.
* **Kesan Sosial:** Bagaimana masyarakat terkesan dengan kecurian identiti.
* **Analisis Kod:** Contoh kod Python dan SQL yang sering digunakan hacker.

## 🛠️ Teknologi Digunakan
* HTML5
* CSS3 (Custom Styling)
* Font Awesome (Icons)
